# VR 手柄控制指南

## 确认键（点击按钮）

### Quest 3 手柄确认键

在 Quest 3 上，用于点击 UI 按钮的确认键通常是：

1. **扳机（Trigger）** - 最常用
   - 位置：手柄前方的扳机按钮
   - 操作：按下扳机进行点击

2. **A 按钮**（右手）或 **X 按钮**（左手）
   - 位置：手柄顶部的按钮
   - 操作：按下按钮进行点击

3. **确认按钮**（如果配置了）
   - 通常是手柄上的主要确认按钮

### 如何测试

1. 运行场景
2. 使用手柄射线指向按钮
3. 尝试按下：
   - **扳机**（最可能）
   - **A/X 按钮**
   - 其他按钮

4. 如果按钮有反应（颜色变化或功能触发），说明那个就是确认键

## 手柄操作总结

### 基本操作

- **指向**：移动手柄，射线会跟随
- **点击**：按下扳机（或 A/X 按钮）
- **选择输入框**：指向输入框并按下扳机
- **输入**：使用虚拟键盘点击数字按钮

### 虚拟键盘操作

- **数字按钮**：点击输入数字
- **点按钮（.）**：输入小数点
- **Del 按钮**：删除最后一个字符
- **Clear 按钮**：清空所有输入
- **OK 按钮**：确认输入并关闭键盘

## 故障排除

### 如果按下扳机没反应

1. 检查 XR Ray Interactor 设置：
   - Interaction Manager 已连接
   - Ray Origin Transform 已设置
   - UI Interaction 已勾选

2. 检查按钮设置：
   - 按钮的 Raycast Target 已启用
   - EventTrigger 已添加

3. 查看 Console 是否有错误

### 如果字体还是显示 T 字

1. 重新运行场景（代码已更新）
2. 检查 Console 日志
3. 如果使用 Prefab，可能需要手动修改 Prefab

## 推荐设置

对于 Quest 3：
- **确认键**：扳机（Trigger）
- **字体**：Unity Text（已自动设置）
- **射线距离**：10 米或更大
